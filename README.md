# Extension-Plus
A mod focused on bringing new content to the game, adding turrets, factories, units, etc.


The mod has:

- 15 Factories
- 3 Conveyor
- 4 Solar Panel
- 3 Battery
- 3 Reactors
- 6 Drills
- 1 Overdrive Projector
- 5 Itens
- 2 Turret
